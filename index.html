  • Swift

    Python is great but I much prefer Swift. This is Mainly because of the following reasons.

      Swift is:

    1. Type Safe
      • it has great type inference
      • it prevents all classes of programming errors to do with dynamic typed languages
      • it has optionals to ensure that any data that may not have a value is dealt with easily

    2. Modern
      • it has features like Generics, Closures, Protocols and extensions on custom and builtin types. It also recently added support for async/await
      • it is open sourced with feature proposals being reviewed every day and new versions being released every year

    3. Fast (well Swift rather)
      • It checks code at compile time helping devs to code faster and prevent runtime errors
      • it has a incredibly fast runtime and can run along side existing C, C++ and Objective-C code
